I downloaded the Adobe LiveMotion Public Beta the other day. It's pretty cool... Kinda like Macromedia's Flash 4, but Adobe-ized. Actually, that's exactly what it is; it exports it's animations as a real Flash 4 file. And, the cool part is that it's interface is very much like AfterEffects, a program that I got to use quite often and was able to do some pretty cool things with, so I was able to get started pretty fast, even without the manual. Pretty cool eh?

I started working on my personal site using nothing but LiveMotion. Right now, I have a simple menu system working and you can go to it now.
